# **Tokyo City Wars Replication**
## **Overview**
**Tokyo City Wars Replication** is a Java-based card game project I did in spring 2024, inspired by the popular game [City Wars: Tokyo Reign](https://store.steampowered.com/app/1689090/CITY_WARS_TOKYO_REIGN/).

- **Phase 1**: Command Line Interface (CLI)
- **Phase 2**: Graphical User Interface using JavaFX and Scene Builder

## ✨ Features

- **Multiple Game Menus** - Intuitive navigation through different game screens
- **Drag-and-Drop Gameplay** - Seamless card placement on the game board
- **Multiple Game Modes** - Different ways to play and enjoy the game
- **Game History** - Track your matches and performance
- **Account Management** - User registration and login system
- **SQL Database Integration** - Persistent storage for player data and game history
- **Interactive UI** - Modern interface built with JavaFX and Scene Builder

## 🛠️ Technology Stack

| Technology | Purpose |
|------------|---------|
| **Java** | Core game logic and OOP implementation |
| **JavaFX** | Graphical user interface |
| **Scene Builder** | UI layout design |
| **SQL** | Database management for accounts and history |
